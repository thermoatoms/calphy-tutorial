# calphy-tutorial

Tutorial for [calphy](https://calphy.org), a Python library and command line tool for calculation of free energies.

## Installation

Create a conda environment from the provided [environment.yml](environment.yml):

```bash
conda env create -f environment.yml
conda activate calphy
```

## Running the tutorial

Start Jupyter Lab and open the notebook:

```bash
jupyter lab CuNi.ipynb
```
